float

float optionen
  • float: left; = platziert das element links
  • float: right; = platziert das element rechts
  • float: none; = default, keine platzierung

floats sorgen dafür, das wir spanelemente links und rechts platzieren können.

Ein element, dem ein float in css zugewiesen wurde, ist kein block-element mehr, nimmt also keinen resevierten platz mehr auf der seite ein.

mit float: left; platzieren wir elemente auf der linken seite (grüne box)

mit float: right; platzieren wir elemente auf der rechten seite (rote box)
