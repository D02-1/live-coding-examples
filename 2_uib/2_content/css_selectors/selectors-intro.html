Selektoren

in css nutzen wir selektoren um das element, das wir stylen wollen anzuwählen, ein css kommando ist wie folgt aufgebaut:

  • [element] { = der selektor
  • [eigenschaft]: [wert]; = die option
  • } = das ende des kommandos

mit komma getrennt, können wir mehrere selektionen das selbe komando geben

beispiel:

Hallo Welt

Noch einmal Hallo Welt

mit sternchen (der universalselektor) können wir alles selektieren

wir können auch direkt elemente ohne klasse oder id verändern

Ich bin ein fetter text
