Text transfomieren

Mit text-transform: ; können wir definieren, wie sich die groß und klein schreibung eines textes verhält, unabhängig davon, wie wir den text im HTML geschrieben haben

UPPERCASE

text-transform: uppercase; sorgt dafür das alle buchstaben innerhalb des elementes groß geschrieben werden

Hallo Welt ddddddddd DDDDDD

lowercase

text-transform: lowercase; sorgt dafür das alle buchstaben innerhalb des elements klein geschrieben werden

Hallo Welt HHHHHHHH dddd

Capitalize

text-transform: capitalize; sorgt dafür das jedes wort innerhalb des elementes mit einem großbuchstaben anfängt

hallo welt ddddd ddddd
