Default Table

Eine tabelle erstellen wir mit dem element "table", dort können wir dann "tr" für eine neue zeile und dort drin "td" für neue spalten nutzen

A B

Tabellen Header

Um einer tabelle eine kopfzeile zu geben, nutzen wir das th element, innerhalb einer eigenen zeile (tr)

First NameLast Name

JamesBond

Fluide Tabellen & Tabellen Titel

Wenn wir die größe von spalten ändern, ist der inhalt der "th", also des table headers zentriert

Mit "caption" können wir der tabelle einen titel geben, das caption element sollte immer das erste kind von "table" sein, also über "tr"

Name Table
First Name Last Name

Tabellen Zellen Padding

Wenn wir padding haben wollen, müssen wir die Table data und table header elemente ansprechen

Tabellen zellen abstand

Mit dem css befehl border-spacing: ; können wir angeben wie groß der abstand zwischen den tabellen zellen ist.

First Name Last Name
Max Mustermann
James Bond

Tabellen spalten zusammenfügen (col-span)

Mit dem html attribut colspan="" können wir definieren wie viele spalten eine spalte breit ist.

First Name Last Name
Max Henry Mustermann
James Theodore Bond

Tabellen Zeilen zusammenfügen (row-span)

Um zellen in zeilen zu kombinieren, könenn wir das attribut row-span="" benutzen.
