Formulare

Beispielformular

ein formular hat 2 essentielle attribute, action="" gibt den pfad zu unserer validierung/auswertung an und method="" die methode, mit der wir die daten bearbeiten, also GET (daten anfragen) und POST (daten senden)

Bei wichtigen, nicht öffentlichen daten, wird immer POST benutzt, da man dann, im browser den datenverkehr nicht nachvollziehen kann.

das element FORM wird benötigt um eine formular zu erstellen

Ein formular enthält immer zusammengehörige informationen die wir input feldern erhalten wollen




Google Beispiel

Input typen

Text

Das attribut placeholder="" definiert, was angezeigt werden soll, wenn das input feld leer ist.

Das attribut maxlength gibt an, wieviele zeichen in das input-feld geschrieben werden dürfen

Password


E-Mail


Number

mit min und max können wir die kleinste, und größte auswählbare zahl angeben

Range

mit value="" können wir angeben welchen wert der range slider zwischen 0 und 100 hat

mit step="" können wir die schritte angeben, in der der range-slider geslided werden kann

Textarea

ein textarea element unterscheidet sich von input elementen, da man hier den gewünschten text direkt zwischen die element tags eingibt

mit cols="" geben wir an wie breit das element sein soll und mit rows="" geben wir an, wie hoch das element sein soll (in zeichen und zeilen)

Buttons

Ein submit button ist dafür da, ein formular abzuschicken

ein reset button ist dafür da, ein formular zurück zu setzen

Nicht-Formular Buttons

ein button des types button ist dafür da events in javascript auszuführen


Der button über diesem text ist ein equivalent zu dem folgenden button, nur das man hier den text direkt zwischen die elements tags schreibt

Checkbox

wir können mehrere checkboxen zur auswahl stellen, der benutzer hat dann die wahl.





Radio

Bei radio buttons können wir immer nur ein element pro gruppe auswählen, eine gruppe trägt immer den selben wert bei name=""

mit dem attribut "checked" können wir definieren, was von anfang an ausgewählt ist, sowohl bei checkboxen, wie auch bei radio buttons




Select

Datalist

Um eine autocomplete-datalist zu erstellen brauchen wir ein input-element mit einem list attribut, das die selbe id hat, wie unsere datalist

wir können sowohl die value, wie auch einen angezeigten wert eingeben, um 2 zeilen text in die auswahl zu bringen


Color

Date

Formularattribute

Ein feld mit dem attribut "readonly" kann nur gelesen, kopiert und selektiert werden, der inhalt kann aber nicht verändert werden


Ein feld mit dem attribut "required" MUSS ausgefüllt werden, damit das formular abgeschickt werden kann


Ein feld mit dem attribut "disabled" kann selektiert und gelesen werden, aber es ist keine weitere interaktion möglich, es erscheint grau, weil es "deaktiviert" wurde.
