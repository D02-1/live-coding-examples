Padding und Margin

Der default wert von padding und margin ist 0; allerdings haben elemente wie p oder h1 ein vorgegebenes margin oder padding

Mit padding und margin können wir abstände innerhalb und zwischen elementen angeben

  • padding = abstand vom inhalt zum rand des elements
  • border = der stylebare rand des elements
  • margin = abstand vom element zum nächsten element
