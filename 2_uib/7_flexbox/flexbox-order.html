order

mit order: ; können wir die angezeigte reihenfolge von elementen verändern, egal wie sie in HTML sortiert sind.

der standardwert von order: ; ist 0

1 (0)
2 (-1)
3 (3)
4 (2)
5 (-2)

5 (-2)
2 (-1)
1 (0)
4 (2)
3 (3)

Wenn elemente die gleiche order: ; nummer haben, werden sie in der reihenfolge angezeigt, in der sie im html code stehen.

1 (0)
2 (-1)
3 (3)
4 (2)
5 (-2)
6 (-1)
